## A base construção Web

```js
A base da construção da web se baseiam nas linguagens Html , Css e linguagens de programação.
```

# Html 

```js
O html não é uma linguagem de programação ele é uma linguagen de hyper texto .

O html vai dar a estruturaão necessaria para seu codigo.

-Hypertexto->Links que conectam as paginas 

-Marcação -> Detalhamento através de TAGs

Exemplos : 

<html>,<body><head>...etc

```

# Css 

```js 

Estilização visual : Define a aparência dos elementos como cores,fontes tamanhos e estilos de bordas .

Layout e Posicionamento : Organiza o layout da página ,controlando como os elementos são dispostos e como se comportam em diferentes tamanhos de tela (responsividade)

Responsividade : Permite que as páginas se adaptem a diferentes dispositivos e tamanhos de tela , como desktops , tablets e smartphones.

Manutenção e Consistência : Fcilita a manutenção do estilo através da separação da apresentação e do conteúdo , permitindo uma estilização consistente e reutilizável em toda a aplicação.

```

# Java Script 

```js 

Linguagem de Programação

Não é responsabilidade do html ou CSS dar dinamismo ou validar informações. 

Segue uma estrutura (sintaxe)  própria.

interatividade e Dinamismo: JavaScript é o que transformou uma página estática em uma experiência viva : cliques ,animações ,feedback imediato ,estados que mudam sem recarregar a pagina .

Manipulação do DOM(Document Object Model):O DOM  é a ponte entre JavaScript é o HTML real que o navegador renderiza. Manipular o DOM é  , na prática ,alterar a estrutura ,conteúdo e o estado visual da aplicação em tempo real .

Validação de Formulários : Validar no frontend  não é sobre  "impedir o envio ",mas sobre orientar o usuário antes do erro acontecer .

Comunicaçao Assincrona : A comnunicação assíncrona permite buscar e enviar dados sem travar a interface , usando fetch ,async ;await e APIs externas .
```
