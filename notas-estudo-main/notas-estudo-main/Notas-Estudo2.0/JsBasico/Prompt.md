## Prompt 

```js 
Ele irá nos permitir realizar duas ações :

Mostrar uma pergunta 

Receber uma resposta do usuário 

Detalhe importante : Toda resposta do prompt é texto (string),mesmo que o usuário digite um número.

Exemplo :
```js
const variavel = (prompt("string:):"))
```

Para que o Prompt não retorne texto precisamos adicionar um elemento chamado Number 

Exemplo :

```js 
const variavel = Number(prompt("Por favor digitar um numero."))
```
```
