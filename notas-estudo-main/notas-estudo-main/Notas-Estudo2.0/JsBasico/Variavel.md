## Variavel

```js 

A variavel não é apenas uma caixinha.Ela é uma referência a um valor na memória.Serve para guardar estado permitindo que o programa lembre de algo ao longo do tempo.Sem variáveis ,todo o programa seria estático.
```

# 3 formas de declaração 

```js 

var: declara uma variável com escopo de função ou global ,podendo ser redeclarada ,tem hosting permissivo (Mas a var nao é recomendavel ser ultilizada por ser um erro.)

let : declara uma variável com escopo de bloco ,que pode ser retribuída ,mas não redeclarada no mesmo escopo .

const : declara uma variável com escopo de bloco que não pode ser reatribuída ,apenas seu conteúdo interno pode mudar (se for objeto/array)

```
