## if/else

```js 

O if e else são estruturas condicionais que permitem executar diferentes blocos de códigos com base em condição ou expressão booleana

Exemplo :
 
let cocupado = 1

if (ocupado ==1){
    console.log("Ocupado")
}

else {
    console.log("Livre")
}

```

# if / Else if / Else 

```js 
Além disso ,tembém é possível ultilizar múltiplas estruturas if aninhadas para verificar  várias condições consecutivas ,ou usar a estrutura else if para adicionar condições indetermediarias .

Exemplo :

let ocupado = 2

if (ocupado ==1){
    console.log("Ocupado")
}

else if (ocupado ==2) {
console.log("Reservado")
}

else {
    console.log("Livre")
}

```
