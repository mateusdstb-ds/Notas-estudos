## introdução


```js 

-React é uma biblioteca JavaScript para construir interface de usuário .

-Desenvolvido pelo facebook

-facilita a criação de componentes reutilizáveis.

-Usa uma abordagem declarativa para criar interfaces.
```

# Components 

```js 

-Em React , a interface de usúario é dividida em pequenas partes reutilizáveis chamadas components .

-Cada componente é responável por renderizar uma pequena e específica parte da UI.Isso facilita a manutenção e ea escalabilidade do código .

```
